# Corporate HR & Sales SQL Analytics Pipeline

## 🚀 Project Overview
This project showcases a comprehensive collection of **30 structured SQL queries** designed to clean, manipulate, and extract business insights from relational databases. The scripts tackle real-world scenarios across Human Resources (HR) performance tracking and Sales operations.

## 🗂️ Core Concepts Demonstrated
Through these 30 queries, I have practically applied and mastered the following relational database concepts:
* **Advanced Filtering:** Pattern matching using `LIKE` and wildcards (`%`, `_`).
* **Table Connections:** Linking multiple datasets using structural `JOIN` operations.
* **Subqueries:** Writing nested queries using Single-value and Multi-value (`IN` operator) logic.
* **Data Integrity:** Detecting duplicate records using `GROUP BY` + `HAVING` clauses and handling missing data via `COALESCE`.

## 🛠️ Project Structure
* **`database_setup.sql`**: Contains the table definitions, structural schemas, and sample data insertions.
* **`analytics_queries.sql`**: Features the final 30 optimization queries fully documented with business context explanations.
